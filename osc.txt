## osc usefull commands

  osc co home:unixmonk3y
  checkout home project

  osc mkpac package_name
  create package

  osc ci -m "message"
  comit changes

  osc add file
  add file

  osc addremove
  add and remove files :)

  osc importsrcrpm file
  import spec file from src rpm

  osc copypac
  copy package

  osb build
  build the package
