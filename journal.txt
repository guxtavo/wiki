# tips

## How to record a screencast

    $ record_audio      # Control+c when finished
    Ctrl+Alt+Shift+r    # repeat shortcut to stop

## How to put together audio and video:

    ffmpeg -i video.webm -i audio.ogg -c:a copy -c:v copy output.webm

## How to add annotation to a video

    # edit bin/fix_video and set text and timing
    fix_video video.webm

## How to get blocks of text with awk

    awk '/block of text/,/^$/' file
    or
    bawk block file

## How to use authentication on curl

    curl -O -u me:mypass "https://apibugzilla.suse.com/..."

## How to fix html files in vim

    :s/<[^>]*>/\r&\r/g
    :g/^$/d

## How to delete all lines containing a certain string in vim?

    :g/sshd/d

## How to remap capslock to control:

    setxkbmap -option ctrl:nocaps

## How to artificially create delay in tc

    sudo tc qdisc add dev tun0 root netem delay 999ms
    sudo tc qdisc del dev tun0 root netem
